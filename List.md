# ADT
- 원소들의 순서가 존재
- 같은 값의 중복 허용
- 위치(index 또는 position) 기반 접근 가능.

## 기본 연산
- create()  : 빈 리스트 생성
- isEmpty() : 비어 있는지 확인
- size()    : 원소 개수 반환
- claer()   : 모든 원소 제거
 
## 접근 연산
- get(i)    : i 번째 원소 반환
- set(i, x) : i 번째 원소를 x로 변경

## ADT vs 구현
- 리스트는 이런 연산을 제공 | 배열 기반 / 연결 리스트 기반
- 인터페이스 정의			| 메모리 구조 정의
- 추상적 개념				| 구체적 자료구조

## 요약 
- 순서가 있는 데이터 집합을 다루는 추상 자료형이다
- 위치 기반 접근과 삽입 / 삭제 연산을 제공한다.
